# EThing UI


web app + mobile app + Electron app

based on [Quasar App](https://quasar-framework.org/) (V1).

## Installation

```
git clone https://github.com/e-thing/ething-quasar.git
cd ething-quasar
npm install
```

## Development

```
# run development server (with default theme)
$ npx quasar dev
```

## Build

```
# build for production
$ npx quasar build
```
